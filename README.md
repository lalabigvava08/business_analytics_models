# business_analytics_models
Business analytics case study including customer churn analysis, regression forecasting, and linear programming optimization using Excel and statistical modelling.


## Problem 1: Customer Churn Analysis

A telecom customer dataset was analysed to identify factors influencing customer churn.

### Key Findings
- High **customer service calls** strongly correlate with churn.
- Customers with **international plans** show significantly higher churn rates.
- Customers with **voicemail plans** show lower churn.

### Business Recommendations
- Proactively contact customers with frequent support calls.
- Review international plan pricing and services.
- Promote voicemail services to improve retention.



## Problem 2: Regression Analysis and Sales Forecasting

A regression model was developed to analyse factors affecting clothing sales.

### Variables
- Holiday Sale Dummy
- Recession Dummy
- Trend Index

### Key Results
- Holiday sales significantly increase sales performance.
- Sales show a positive long-term growth trend.
- Forecasting shows a **72% probability of exceeding 1000 units in Year 24**.


## Problem 3: Linear Programming Optimisation

An optimisation model was built using **Excel Solver** to minimise supplier costs while meeting component requirements.

### Objective
Minimise procurement cost from three suppliers.

### Optimal Solution
- Supplier 1: 125 units
- Supplier 2: 500 units
- Supplier 3: 500 units

Total Cost: **£16,500**

Sensitivity analysis showed that demand increases above **14% make the model infeasible** due to supplier capacity constraints.
